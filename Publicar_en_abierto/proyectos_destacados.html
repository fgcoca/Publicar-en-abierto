Proyectos destacados

Proyectos destacados

Algunos ejemplos son:

  • Proyecto RepRap: Una impresora 3D libre, autoreplicante.
  • Proyecto Clone Wars: Documentación en español dentro de RepRap.
  • OpenRISC: grupo de desarrolladores trabajando para producir un CPU RISC libre de muy alto rendimiento.
  • RISC-V: arquitectura de conjunto de instrucciones libre.
  • Arduino: plataforma libre de computación física basada en una tarjeta I/O y un IDE.
  • Robótica de código abierto. ¡¡¡ MASAYLO !!!
  • Ventilador de código abierto (respirador para UCI).
  • gEDA: suite completa GPL de herramientas de automatización de diseño electrónico.
  • Tarjeta open FPGA Alhambra II: Sintetizar hardware de forma sencilla utilizando herramientas open source.
    • Comunidad abierta FPGAwars: web donde se publican talleres y tutoriales.
    • Grupos para contactar con otros usuarios.
    • Repositorios GitHub
    • Lideres: @obijuan_cube y @EladioDM
  • Raspberry Pi
  • Telefónos móviles de código abierto: No basados en Android porque usa muchos componentes propietarios. Sistemas operativos como Firefox, Debian o Ubuntu
